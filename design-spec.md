# design-spec.md — Moor & Milk（架空ベーカリーEC）

> このファイルが**唯一の正（source of truth）**。UIの生成・修正は必ずここを入力にする。
> 想定する構造・トーン: スコーン/ミルクティー系の小規模ベーカリーECの一般的なレイアウト。
> ※架空ブランド **Moor & Milk** 専用の設定。実在ブランド名・実商品名・実画像は使用しない。

## 0. コンセプト（架空ブランド）
- **ブランド名**: Moor & Milk（ムーア アンド ミルク）
- **タグライン**: 「焼きたてスコーンと、ほっとするミルクティーのお店。」
- **世界観**: イギリスの片田舎（moor=荒野）の素朴な焼き菓子 × 日本の丁寧なミルクティー。クリーム×白基調の素朴でミニマルなトーン。

## 1. デザイントークン
### カラー
| 用途 | 値 | 備考 |
|---|---|---|
| 背景 (base) | `#FFFFFF` | body bg = rgb(255,255,255) |
| 文字 (foreground) | `#1A1A1A` | 真黒すぎを避け #1A1A1A |
| クリーム (section alt) | `#F7F3EC` | セクション交互の淡い生成り |
| アクセント（News帯/CTA） | `#E08A3C` | News の "Tasty Gifts" 帯が温かいオレンジ |
| アクセント淡 | `#F0D8B8` | バッジ・ボーダーの淡色 |
| SOLD OUT グレー | `#9CA3AF` | 売切れカードの半透明グレー |

### タイポグラフィ
| 役割 | フォント | 用途 |
|---|---|---|
| 英語ディスプレイ見出し | **League Gothic**（縦長コンデンス）| "About Us" "Seasonal" 等のセクション英語ラベル |
| 日本語本文/見出し | Zen Kaku Gothic New / Yu Gothic 系 sans | 日本語サブタイトル・本文 |
| 補助セリフ | Josefin Slab 系 serif | 一部の装飾見出し |

- 本文 base: 16px / 行間ゆったり
- セクション見出し: 英語ディスプレイ ~24–48px + 直下に日本語サブタイトル（小さめ・グレー寄り）

### レイアウト
- コンテンツ最大幅: ~1080–1200px 中央寄せ、左右に広めの余白
- 商品グリッド: **デスクトップ4列 / タブ2–3列 / モバイル1–2列**
- セクション間: 大きめの縦余白（ゆったり）。ページ全長は長い（縦スクロール型LP）
- ブレークポイント: Tailwind 既定（sm 640 / md 768 / lg 1024 / xl 1280）

### 署名的モチーフ（必ず踏襲）
> **セクション見出し = 「英語の大きな League Gothic ラベル」＋「直下の小さな日本語サブタイトル」**
> 例: `SEASONAL` の下に「今だけ出会える季節限定商品」。これがサイト全体の視覚的アイデンティティ。

## 2. セクション順序（上から）= コンポーネント分割
この並び順に対応させる。各 = 1コミット。

1. **Header / Nav**（固定 or 上部）: ロゴ（Moor & Milk）＋ナビ `Top / About / Shop / Seasonal / Signature / Journal / Contact` ＋カートアイコン
2. **Hero**: フルブリードの大きな商品写真1枚（スコーン）。テキストは最小（タグライン）。カルーセル風だが初期実装は単一画像でも可
3. **About**: 左テキスト（ブランド説明）＋右画像の2カラム。見出し `ABOUT US / ムーア アンド ミルクのこと`
4. **News**: 横長バナー（オレンジ系）"Seasonal Gifts" 訴求。見出し `NEWS / 注目のニュース`
5. **Seasonal**: 季節限定の特集。画像＋テキストの特集カード 1–2枚。見出し `SEASONAL / 今だけ出会える季節限定商品`
6. **Product Grid（Items）**: 商品カードの4列グリッド（12–16枚）。一部 **SOLD OUT** バッジ＋半透明。見出し `ITEMS / 商品ラインナップ`
7. **Journal（Bakers Tips相当）**: ブログ記事カードのグリッド。見出し `JOURNAL / 暮らしを楽しむ読みもの`
8. **Footer**: 大きなブランドロゴ＋ナビ＋SNS＋コピーライト

## 3. コンポーネント仕様
- **SectionHeading**: props `{ en: string; ja: string }` → League Gothic大見出し＋日本語サブ。全セクションで再利用
- **ProductCard**: props `{ name; price; image; soldOut?: boolean }`。正方形画像＋商品名＋価格、soldOut時はグレーオーバーレイ＋"SOLD OUT"バッジ
- **BlogCard**: props `{ title; date; image; excerpt }`
- **Nav**: ロゴ＋リンク群＋カート。モバイルはハンバーガー（初期は簡易でも可）

## 4. ダミーデータ（架空商品）
スコーン/ミルクティー系の架空ラインナップ例（実名を避けた創作）:
- バターミルクスコーン / 紅茶とオレンジピールのスコーン / チョコ＆アーモンドスコーン / 全粒粉スコーン
- いちごジャムのクロテッドスコーンサンド / 抹茶とホワイトチョコのスコーン
- 焦がしバターフィナンシェ / レモンマドレーヌ / クッキーアソート / 焼きチョコケーキ
- ロイヤルミルクティー（茶葉）/ ほうじ茶ミルクティー / オリジナルトートバッグ
- 価格は ¥XXX 形式のダミー。数点を SOLD OUT に
