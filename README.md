# Moor & Milk — スターターキット

AI研修 **Session 9（Claude Code × CLI 入門）** で使う、架空ベーカリーEC「**Moor & Milk**」のスターターコードです。
これは**機能を追加する前のベースライン**です。ここを出発点に、あなたが作りたい機能を Cursor + Claude Code で追加していきます。

> スコーンとミルクティーのオンラインショップ、という設定の架空サイトです。実在のブランド・商品ではありません。

---

## このリポジトリの使い方（受講者の方へ）

1. このページ右上の緑色の **「Use this template」** ボタンを押して、**自分用のリポジトリ**を作成します。
   （これであなた専用のコピーができ、元の教材リポジトリを汚さずに自由に編集できます）
2. 作成した自分のリポジトリを Cursor で開きます（次の「セットアップ」参照）。
3. Cursor の内蔵ターミナルから Claude Code を起動し、機能を追加していきます。

> 「Use this template」が見当たらない場合は、`git clone` または **Code → Download ZIP** でダウンロードしてもOKです。

---

## セットアップ

### 必要なもの
- **Node.js 20 以上**（推奨: 22）
- **Cursor**（または VS Code）
- **Claude Code**（Cursor の内蔵ターミナルから使います）

### 起動手順

```bash
# 1. 依存関係をインストール
npm install

# 2. 開発サーバーを起動
npm run dev
```

ブラウザで **http://localhost:3000** を開くと、Moor & Milk のトップページが表示されます。
コードを保存すると自動でリロードされます。

その他のコマンド:

```bash
npm run build   # 本番ビルド（エラーがないか確認したいとき）
npm run lint    # コードチェック
```

## Cursor + Claude Code で機能を追加する流れ

1. Cursor でこのフォルダを開く
2. 内蔵ターミナル（`Ctrl/Cmd + J`）を開く
3. `claude` と入力して Claude Code を起動
4. やりたいことを日本語で伝える（例:「商品をカテゴリで絞り込めるようにして」）
5. 提案された差分（diff）を確認し、`npm run dev` で動作を確かめる

> Claude Code は、まず作業計画を提示してから実装します。差分を見て納得してから進めましょう。

## チャレンジ課題（自分で作ってみよう）

研修のデモで扱った3つの機能です。**答えは入っていません**。Claude Code に相談しながら、自分の手で実装してみてください。

1. **商品検索 / カテゴリ絞り込み** — 一覧（ProductGrid）にキーワード検索とカテゴリフィルタを追加する

## このプロジェクトについて（技術メモ）

- **フレームワーク**: Next.js 16（App Router）
- **UI**: React 19 / Tailwind CSS v4 / TypeScript
- **商品・記事データ**: `src/data/products.ts` / `src/data/posts.ts`（静的データ。ここを編集すると表示が変わります）
- **コンポーネント**: `src/components/`（Header, Hero, ProductGrid, ProductCard, Seasonal, Footer など）
- **デザイン定義**: `design-spec.md`（このサイトの「正」となるデザイン仕様。UIを直すときの入力に使えます）

### Claude Code 利用者向けのファイル
- `AGENTS.md` … **Next.js 16 は破壊的変更が多い**ため、AIに「学習データの古い書き方を使わない」よう促すメモです。Claude Code がこれを読みます。
- `CLAUDE.md` … `AGENTS.md` を参照する設定です。

これらは消さずに残しておくと、Claude Code がこのプロジェクトに合った書き方をしてくれます。
